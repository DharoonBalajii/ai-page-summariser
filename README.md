# ✨ AI Page Summariser

![Version](https://img.shields.io/badge/version-1.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Platform](https://img.shields.io/badge/platform-Chrome_Extension-yellow.svg)

AI Page Summariser is a powerful, lightweight Chrome Extension that transforms any long, boring webpage into a concise, beautiful summary using Google's cutting-edge **Gemini AI**.

Stop wasting time reading endless articles. Get the gist instantly, ask follow-up questions, and even have it read aloud to you!

## 🚀 Features

- **⚡ Instant Summaries**: Click a single button to perfectly summarize the active webpage.
- **🎯 Customizable Styles**: Choose how you want your information:
  - Bullet Points
  - Gist
  - Explain Like I'm 5
  - Executive Summary
  - Action Items
- **💬 Chat with the Page**: A built-in Q&A chat allows you to ask Gemini direct questions about the page's content.
- **🖱️ Right-Click Context Menu**: Highlight any text on the internet, right-click, and select "Summarise with AI" for an instant, beautiful floating summary.
- **🔊 Read Aloud**: Built-in Text-to-Speech allows you to listen to your summaries on the go.
- **💾 Export**: Copy to clipboard or download as a `.txt` file instantly.
- **⏱️ Time Saved Calculator**: See exactly how many minutes of reading time you saved!

## 🛠️ Installation

1. Clone or download this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions`.
3. Enable **Developer mode** (toggle switch in the top right corner).
4. Click the **Load unpacked** button in the top left.
5. Select the folder containing this repository's files.
6. The extension is now installed! Pin the ⭐ icon to your toolbar.

## 🔑 Setup

To use the extension, you need a free Google Gemini API key:
1. Click the extension icon in your browser.
2. Click **"🔑 Get API Key from Google AI Studio"** to generate a free API key.
3. Paste the key into the extension and click **Save**.
4. You are ready to go! Your API key is stored securely and locally on your browser.

## 🛡️ Privacy & Security
This extension runs entirely in your browser. Your API key is saved to your local Chrome storage and is never transmitted anywhere except directly to Google's official API servers. No tracking, no external servers, and completely transparent source code.
